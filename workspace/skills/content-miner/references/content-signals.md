# 7 Content Mining Signals (Universal)

## 1. Aha Moments / Realizations
"We just realized that..." / Sudden insight, reframe

## 2. Conflicts / Problems Solved
Problem faced + solution applied (before/after)

## 3. Contrarian / Unpopular Opinions
Challenges norms, "best practices", conventional wisdom

## 4. Systems / Processes
Repeatable process, framework, tool setup explained

## 5. Milestones / Wins
Numbers, metrics, measurable achievements (even small)

## 6. Vulnerability / Struggles
Honest admissions: difficulty, doubt, failure, frustration

## 7. Before/After Metrics
Explicit comparison across time ("from X to Y")
