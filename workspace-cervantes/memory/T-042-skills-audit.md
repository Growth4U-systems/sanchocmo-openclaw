| Skill | Size | Acción |
|-------|------|--------|
| **seo-content** | 57KB | Mover prompts a references/prompt.md |
| **email-sequences** | 55KB | Crear references/, mover templates |
| **content-atomizer** | 55KB | Crear references/, separar por plataforma |
| **keyword-research** | 54KB | Mover methodology a references/ |
| **newsletter** | 48KB | Crear references/, separar secciones |
| **lead-magnet** | 43KB | Mover examples a references/ |
| **direct-response-copy** | 41KB | Mover frameworks a references/ |
| **comic-ui-system** | 26KB | Aceptable (es un design system) |
| **pricing-strategy** | 22KB | Merge con pricing-hooks primero |
| **thief-marketers** | 21KB | Mover examples a references/ |
| **last30days** | 20KB | Evaluar si los 244 archivos son necesarios |

**Estimación de ahorro**: Si se reestructuran los 10 oversized skills → ahorro de ~300KB de tokens por carga → ~$4.50/MTok ahorrados en un día típico con múltiples invocaciones.
