## Lessons learned (IMPORTANTES)
- **openclaw.json schema es ESTRICTO**: No inventar keys. Leer docs primero.
- **Crons**: `openclaw cron add`, NO editar openclaw.json
- **Channel overrides**: Solo `systemPrompt`, `requireMention` etc. NO `agent`
- **Sub-agentes para CSS**: El sub-agente rompió JS al tocar demasiado. Mejor hacer cambios CSS quirúrgicos uno a uno desde la sesión principal.
- **body::before/after + nav position:fixed**: Cuidado con selectores CSS que sobreescriben position de nav
- **curl > urllib.request**: Para Discord API, curl funciona pero urllib da error 1010
