### Competitor Deep-Dive Template — Iterative Design with Alfonso
- Layout: LEFT column (Strengths + Weaknesses stacked) | RIGHT column (KPIs cards + Rating cards + Reviews 2-col)
- Alfonso preferences discovered through iteration:
  - NO excessive whitespace — dense layout preferred
  - Text large enough to read easily (body-size ~0.85-1.05rem)
  - Bullets with spacing between items (not flat list)
  - Real platform logos (Trustpilot, App Store, Google Play) from Google Favicon Service, base64 embedded
  - Logos in vertical cards: icon top-left + name below, then score row
  - Clear visual separation between sections (borders, shadows, colored backgrounds at 10%)
  - Reviews same font size as S/W column
  - Vertical divider between Positive/Negative reviews
  - Brand colors from client's visual-identity, not hardcoded
  - Links to full docs must be functional (MC URLs)
  - Footer should NOT duplicate KPIs already shown in right column
- Example served at: `https://staging.sanchocmo.ai/mc/brand/example/presentations/competitor-deep-dive.html`
