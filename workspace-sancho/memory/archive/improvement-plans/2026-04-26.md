## Temas analizados

### 1. GTM Stack Orchestration (Clay + ICP + TAM)
**Fuente:** Maja Voje — LinkedIn Post (últimas 2 semanas)

**Resumen del framework:**
1. **ICP + closed-won analysis** → CRM data → LLM extrae firmographics/technographics que predicen deals que cierran
2. **TAM scoring** → score prospects contra closed-won, rankea TAM completo antes de entrar en cadence
3. **Clay como capa de orquestación** → señal detection (1st/2nd/3rd party), AI account research, qualification, tiering, waterfall enrichment, routing
4. **Demand generation** → contenido, outbound, paid targeting hacia Clay list
5. **Self-improving system** → closed-won retrains ICP model; contenido genera 1st-party signals; paid targets tier 1

**Idea aplicable a Growth4U/SanchoCMO:**
- Integrar este framework en `acquisition-metrics-plan` (skill existente)
- Añadir un módulo de **ICP scoring** que use LLM para analizar closed-won del cliente
