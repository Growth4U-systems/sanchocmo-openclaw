## Learnings (captured in SKILL.md)
1. Cover centering: inner div needs `flex:1` for `justify-content:center` to work in flex column parent
2. Logo: NEVER use generic SVG circles — always fetch real logo (scraping web if needed)
3. Niche pairs: each niche = detail slide + campaign slide (immediately after, never separated)
4. Competencia slide early (slide 3-4) — provides context before niches
5. Foundation paths vary per client — always `find brand/{slug} -name "current.md"` first
6. `box-sizing: border-box` required on `.slide` so padding doesn't break viewport
7. Niche descriptions must be RICH: full paragraph with bolds from Foundation, not 1-line summary
8. Positioning data drives ad copy: headlines, CTAs, USPs come from positioning-messaging
9. Nav arrows overlap: add lateral padding to slides or push arrows to edge
10. Large presentations (>15 slides) timeout Escudero — use Python scripts for batch modifications
11. Duplicate slides from incremental edits — need cleanup script after batch operations
