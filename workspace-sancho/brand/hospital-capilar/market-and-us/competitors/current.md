## 1. Insparya (Trustpilot 3.8/5)

| Métrica | Valor |
|---|---|
| Facturación | €20.7M |
| Clínicas España | 5 |
| Clínicas internacionales | 10 |
| Anuncios Meta activos | +170 |
| Foco principal | Cirugías (mayoría gasto Meta) |
| CR7 | **Cofundador** (no embajador) — awareness brutal |
| Tratamientos propios | **ActivePlasma** (PRP), **MesoHAir+** (mesoterapia) |
| Página mujer | Sí — página dedicada alopecia femenina |
| Digital | Diagnóstico gratuito como CTA en todas las páginas. NO quiz interactivo. CTA siempre teléfono/formulario |

### Vulnerabilidades
- "Te tratan como ganado" (reviews)
- Seguimiento nefasto post-cirugía
- Publicidad engañosa según reviews
- **Tienen tratamientos pero NO los posicionan como core** — gasto Meta = mayoría trasplante
- Sin quiz interactivo ni recomendación personalizada
- CTA siempre teléfono/formulario — el quiz de HC es un diferenciador real

### Cómo ganarles
- Mensaje **"tratamientos primero, cirugía si hace falta"**
- Transparencia total en precios
- Seguimiento real con tricoscopia a 3 meses
- No vender cirugía a jóvenes
- Quiz interactivo como diferenciador (ellos no lo tienen)

## 2. Svenson (Trustpilot 1.5/5 — "Muy malo")

| Métrica | Valor |
|---|---|
| Facturación | €17.4M |
| Clínicas | 31 (mix estética) |
| Foco | Estética general, posicionan tratamientos como core |
| Propietario | NK5 (compra reciente, inyección capital) |
| Agencia | **Havas Creative** (nueva) |
| IG | ~9K seguidores (muy bajo para su escala) |
| Cirugía desde | €2,575 (más barato que HC) |

### Movimientos recientes ⚠️
- Campaña sept 2025: **"No te conformes con recordar tu pelo"** — humor, digital-first, fase 2 TV
- **Están despertando y se mueven rápido**
- Posicionan tratamientos como core (al revés que Insparya)

### Vulnerabilidades
- **No son médicos** (esteticistas)
- Trustpilot **1.5/5** — reviews tipo "pagué €3,500, cero resultados"
- Denuncia OCU
- Presión agresiva en primera visita
- Instalaciones antiguas
- Problemas de concurso de acreedores (histórico)
- **Sin credibilidad médica para respaldar su posicionamiento de tratamientos**

### Cómo ganarles
- Reforzar **"médicos de verdad"** — ellos posicionan tratamientos pero sin credibilidad médica
- Tricoscopia + analítica vs cosmético
- Centros modernos
- Mejor precio: **HC €820 + diagnóstico vs Svenson €1,300-2,600**
- Su Trustpilot (1.5/5) vs HC (4.8/5) = argumento demoledor

## 4. Medical Hair

| Métrica | Valor |
|---|---|
| Precio sesión | €70 |
| Reviews | 1,570 (4.9/5 — posiblemente falsas) |
| Modelo | Satélite (multi-local) |

### Vulnerabilidades
- Baja calidad, poca personalización
- Reviews dudosas (volumen + puntuación sospechosos)
- Alta rotación de personal
- Sin analítica hormonal
- Modelo satélite = dilución de calidad

### Cómo ganarles
- Calidad médica > precio bajo
- Especialización > generalismo
- No competir en primer precio
- **"Somos médicos, no esteticistas"**

## 5. IMD

| Métrica | Valor |
|---|---|
| Clínicas | 21 |
| Foco | Mujeres, estética (láser, vitaminas) |

### Vulnerabilidades
- Esteticistas con función comercial
- **No son médicos**
- Sin analítica hormonal
- Alta rotación de personal

### Cómo ganarles
- Analítica hormonal real vs estético
- Tratamiento de **causa vs síntomas**
- Captar mujeres con mensaje: **"tu alopecia es hormonal, necesitas analítica"**
- Precio más competitivo

## El Cuadrante Vacío (La Oportunidad)

```
                    Tratamientos como CORE         Tratamientos secundarios
                    ─────────────────────────────────────────────────────
Credibilidad       │                              │                      │
médica ALTA        │  ← HC DEBE ESTAR AQUÍ        │  Insparya             │
                    │  (nadie lo ocupa)             │  (los tiene pero no   │
                    │                              │   los posiciona)      │
                    ├──────────────────────────────┼──────────────────────┤
Credibilidad       │  Svenson                     │  Clínicas estéticas   │
médica BAJA        │  (los posiciona pero sin     │  locales              │
                    │   credibilidad)               │                      │
                    ─────────────────────────────────────────────────────
```

**Nadie ocupa el cuadrante de tratamientos como core + credibilidad médica alta.** Es exactamente donde HC puede entrar.
