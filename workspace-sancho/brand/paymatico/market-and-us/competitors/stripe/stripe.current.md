# Battle Card: Stripe

**Tier**: B | **Type**: Indirect (gigante global, no enfocado en España) | **Updated**: 2026-03-04

---

## Contexto Competitivo

Stripe es el **gigante global de pagos online** (fundado 2010, valuación ~$95B), líder en el mercado de payment gateways para startups, SaaS y e-commerce global. Para Paymático, Stripe es competencia **indirecta** porque:
- **No regulado por Banco de España** (opera bajo licencias de otros países EU)
- **Target global, no España-específico** (carecen de Bizum, métodos locales españoles hasta hace poco)
- **Pricing enterprise/global** (no optimizado para SMB español)
- **Self-service product-led** (no servicio dedicado para modelos complejos)

**Por qué algunos clientes eligen Stripe over Paymático:**
- Brand recognition global (startups VC-backed quieren "usar lo que usan todos")
- API potente y documentación excelente (developer-first)
- Ecosystem completo (Stripe Billing, Connect, Atlas, Radar, etc.)

**Por qué Stripe NO es threat directo para Paymático:**
- No especializados en **franquicias españolas**, **escrow regulado por BdE**, o **cuentas programables con marca blanca**
- Pricing alto para mid-market español (1.5% + €0.25 doméstico, 3.25% extra internacional)
- Customer support deficiente (complaint recurrente en reviews)
- **No tienen expertise en modelos complejos regulados por España** (split payments para franquicias, PSD2 específico, gestores de dinero de terceros)

---

## Quick Profile

- **Fundada**: 2010 | **HQ**: San Francisco, USA | **Equipo**: ~8,000+ empleados
- **Funding**: VC-backed, valuación ~$95B
- **Growth model**: Product-led (developer-first, self-service) + Enterprise sales (grandes cuentas)
- **Regulación**: Licencias EU (vía subsidiarias), **no entidad de pago española**

---

## Their Positioning (Lens 1)

**Claim**: "Financial infrastructure for the internet"  
**Value prop**: Plataforma global de pagos con API potente, ecosystem completo (Billing, Connect, Radar), y scale para startups → enterprises.

**Target**: Startups tech, SaaS, e-commerce global, marketplaces, plataformas.

**Key features**: API world-class, Stripe Billing (suscripciones), Connect (marketplaces), Radar (fraud), Atlas (company formation), +100 payment methods, multicurrency.

**Pricing España**:
- Doméstico: **1.5% + €0.25**
- Internacional: **+3.25%** extra
- Currency conversion: **+2%**
- SEPA Direct Debit: fees más bajos (no public rate)
- Disputas: €15-€20
- **Enterprise**: custom rates (volumen alto, negociable)

---

## External Perception (Lens 2)

**Media**: Líder global de pagos, developer-first, innovación (embedded finance, crypto, etc.).

**SEO**: Dominan keywords globales de pagos, developer docs, contenido educativo extenso.

**Recognition**: Usado por Amazon, Shopify, Uber, Lyft, etc. — brand recognition altísimo.

---

## Customer Reality (Lens 3)

**Rating**: **Mixto** — Trustpilot bajo (~2-3/5, quejas sobre holds/support), Capterra/G2 alto (~4-4.5/5, tech users satisfechos).

**Love**:
- API excelente, documentación world-class
- Ecosystem completo (Billing, Connect, etc.)
- Facilidad de integración para developers

**Hate**:
- **Retención de fondos** (holds de hasta 21 días, freezes de cuentas)
- **Customer support deficiente** (lento, automatizado, no resuelve problemas)
- **Chargebacks/disputas** — percepción de que Stripe no protege bien al merchant
- **Pricing alto** para volúmenes medios (vs competidores españoles)

**Unmet needs**:
- Marca blanca para franquicias
- Cuentas programables con escrow
- Soporte dedicado para modelos complejos

## How to Beat Them

### Their Weakness, Our Strength

**1. Global genérico vs España-específico**
- Stripe: API global, carecen de Bizum, métodos locales españoles (hasta hace poco)
- **Paymático**: Regulado BdE, expertise en España, soluciones para franquicias españolas

**2. Self-service vs servicio dedicado**
- Stripe: Product-led, customer support deficiente
- **Paymático**: Servicio dedicado para modelos complejos (franquicias, escrow, gestores de fondos)

**3. Pricing alto vs pricing ajustado**
- Stripe: 1.5% + €0.25 doméstico, 4.75% internacional
- **Paymático**: Pricing competitivo para mid-market español

**4. API genérica vs cuentas programables**
- Stripe: Potente API, pero no cuentas programables con escrow/marca blanca específica para franquicias
- **Paymático**: Cuentas programables diseñadas para modelos complejos

### Positioning

**"Stripe es para startups globales. Paymático es para franquicias españolas."**

**Messaging**:
> "Stripe tiene API world-class y ecosystem global. Pero si gestionas una red de franquicias en España, necesitas más que API bonita: necesitas cuentas programables con split payments automáticos, marca blanca regulada por Banco de España, y un partner que ENTIENDA modelos complejos. Eso es Paymático."

### What NOT to Compete On

- API/developer experience — Stripe es líder ahí
- Ecosystem (Billing, Connect, Atlas) — Stripe ha invertido años
- Brand recognition global — Stripe es el default para startups VC-backed

**Competir en**: Expertise España + Modelos complejos (franquicias, escrow) + Servicio dedicado + Regulación BdE específica

### Sales Talking Points

1. **"Stripe es global. Nosotros somos España."**
2. **"API bonita no resuelve escrow/split payments/marca blanca para franquicias."**
3. **"Customer support: ellos tienen Trustpilot 2/5, nosotros servicio dedicado."**
4. **"Regulación BdE desde 2013. Stripe no es entidad española."**
5. **"Pricing: 1.5% + €0.25 de Stripe vs nuestro pricing ajustado."**

## Monitoring Triggers

1. Stripe lanza **soluciones específicas para franquicias españolas**
2. Stripe obtiene **licencia Banco de España** (unlikely)
3. Stripe **baja pricing** en España agresivamente
4. Stripe adquiere **fintech española** (ej. MONEI, Paynopain)
