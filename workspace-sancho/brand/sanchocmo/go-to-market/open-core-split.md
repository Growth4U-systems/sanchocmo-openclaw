# Open-Core Split — SanchoCMO
<!-- v1.0 | 2026-03-08 -->

## Principio Rector

El open source debe ser **lo suficientemente bueno para que la gente lo use y recomiende**, pero **no tan completo que nadie pague**. El valor premium no está en limitar features — está en **conveniencia, escala, y soporte**.

---

## 🔓 Open Source (Community / Self-hosted)

### Foundation completa (6 layers)
- Company Brief → Market → Competitors → SWOT → Niche Discovery → Positioning
- **Por qué gratis:** Es el hook. Si la Foundation es buena, se quedan. Limitarla mata la adopción.

### Skills core
- company-context
- business-model-audit
- budget-constraints
- market-intelligence
- competitor-intelligence
- self-intelligence
- swot-analysis
- niche-discovery-100x
- positioning-messaging
- **Por qué gratis:** Sin estos no hay Foundation. Y Foundation es el diferenciador.

### Ejecución básica
- seo-content
- email-sequences
- direct-response-copy
- keyword-research
- **Por qué gratis:** Que puedan crear contenido con lo que Foundation descubrió.

### 1 brand/workspace
- Configuración para 1 empresa
- **Por qué gratis:** El solopreneur self-hosted solo necesita 1.

### Discord bot básico
- Interacción conversacional, hilos, onboarding
- **Por qué gratis:** El producto ES el bot. Sin esto no hay producto.

### CLI + setup local
- Instalación, configuración, documentación completa

### Comunidad
- GitHub Issues + Discord público (soporte peer-to-peer)

## 💎 Pro (Hosted / Premium)

### Cloud hosted
- No montar nada. Invitas al bot a tu Discord → funciona
- **Por qué premium:** Conveniencia. El 80% no quiere configurar servers.

### Multi-brand
- N workspaces/clientes en 1 instancia
- **Por qué premium:** Agencias y consultores pagan por esto sin pensarlo.

### Skills avanzados
- thief-marketers (espionaje competitivo semanal)
- deep-research (investigación multi-fuente)
- social-media-extractor (Instagram, TikTok, Reddit)
- lead-magnet (creación de lead magnets)
- funnel-architect (diseño de funnels)
- content-calendar-planner
- signal-monitor (alertas automáticas)
- acquisition-metrics-plan
- **Por qué premium:** Generan ROI continuo. Monitoreo + inteligencia competitiva recurrente.

### Agentes especializados
- Escudero (ejecución paralela, tareas largas)
- Rocinante (QA, brand review, devil's advocate)
- **Por qué premium:** Multi-agente es infra compleja. El self-hosted tiene solo Sancho.

### Crons + automatización
- Heartbeats automáticos
- Monitoring recurrente (thief-marketers semanal)
- Alertas proactivas
- **Por qué premium:** El valor de "trabaja mientras duermes".

### Visual Identity + Brand Voice
- visual-identity (sistema de diseño completo)
- brand-voice (guía de voz + AI Brand Kit)
- **Por qué premium:** Layer 5 completa. Para quien quiere marca profesional.

### Integraciones premium
- Google Workspace (Gmail, Calendar, Drive)
- Notion API
- Supabase analytics
- Apollo (enrichment)
- Apify avanzado (scraping)
- **Por qué premium:** APIs que cuestan dinero mantener.

### Soporte prioritario
- Respuesta <12h (Growth) / <4h (Scale)
- Calls estratégicas mensuales (Scale)

### Updates automáticos
- Nuevos skills, mejoras, sin `git pull`

### Analytics + dashboards
- Métricas de contenido, waitlist, funnel, reporting

## La diferencia filosófica

**Community = Un CMO que te ayuda a PENSAR**
- Para: developer/founder técnico que disfruta montar cosas
- Valor: Estrategia (Foundation)
- Límite: 1 marca, manual, skills core
- Coste real: Gratis, pero pagas tu API key (~€30-50/mes) + hosting

**Pro = Un CMO que piensa Y EJECUTA por ti**
- Para: founder/marketer que quiere resultados, no configurar
- Valor: Estrategia + Ejecución + Automatización
- Límite: Multi-marca, automático, skills completos
- Coste: €49-199/mes (todo incluido, APIs incluidas)

---

## El punto clave

En self-hosted, el usuario paga sus propias API keys (Anthropic, OpenAI, etc.) + hosting. Eso solo ya son ~€30-50/mes en APIs. El Pro a €99/mes con APIs incluidas es un no-brainer para quien no quiere gestionar infra.
