# Diagnostic Scoring Model

4 dimensions, each 0-10. Lowest score = bottleneck = routing target.

---

## Foundation Score (0-10)

Measures: Do we know WHO we serve and WHAT we say?

| Input | Weight | Score Logic |
|-------|--------|-------------|
| Company context answered (what/want/believe) | 2 | 0 = nothing, 1 = partial, 2 = complete |
| ICP / target segments defined | 2 | 0 = unknown, 1 = rough, 2 = data-backed |
| Competitor landscape mapped | 2 | 0 = unknown, 1 = top 3 named, 2 = analyzed |
| Positioning / messaging exists | 2 | 0 = none, 1 = generic, 2 = per-niche |
| Brand voice defined | 1 | 0 = no, 0.5 = implicit, 1 = documented |
| Pricing strategy exists | 1 | 0 = no, 0.5 = basic, 1 = strategic |

---

## Funnel Score (0-10)

Measures: Does traffic have somewhere to go that converts?

| Input | Weight | Score Logic |
|-------|--------|-------------|
| Landing pages exist (not just homepage) | 2 | 0 = none, 1 = homepage only, 2 = dedicated LPs |
| Analytics tracking in place | 2 | 0 = none, 1 = basic GA, 2 = events + attribution |
| Signup/conversion flow works | 2 | 0 = none, 1 = exists but unoptimized, 2 = tested |
| Lead follow-up process exists | 2 | 0 = none, 1 = manual/slow, 2 = systematic |
| Conversion rate is known | 1 | 0 = unknown, 0.5 = estimated, 1 = measured |
| Email/nurturing sequences exist | 1 | 0 = none, 0.5 = basic, 1 = automated |

---

## Traffic Score (0-10)

Measures: Are people finding us?

| Input | Weight | Score Logic |
|-------|--------|-------------|
| Organic traffic (SEO) | 2 | 0 = none, 1 = <1K/mo, 2 = >1K/mo |
| Social media activity | 2 | 0 = inactive, 1 = posting, 2 = engaged audience |
| Paid advertising running | 2 | 0 = none, 1 = small, 2 = significant |
| Outreach/partnerships active | 2 | 0 = none, 1 = occasional, 2 = systematic |
| Content being produced | 1 | 0 = none, 0.5 = occasional, 1 = regular |
| Referral channel exists | 1 | 0 = none, 0.5 = organic, 1 = formalized |

---

## Revenue Score (0-10)

Measures: Is traffic becoming revenue?

| Input | Weight | Score Logic |
|-------|--------|-------------|
| Conversion rate (traffic to customer) | 2 | 0 = unknown, 1 = <2%, 2 = >2% |
| Unit economics clear (CAC, LTV) | 2 | 0 = unknown, 1 = estimated, 2 = measured |
| Revenue growing | 2 | 0 = declining, 1 = flat, 2 = growing |
| Pricing optimized | 2 | 0 = no strategy, 1 = basic, 2 = tested |
| Multiple revenue streams | 1 | 0 = single, 0.5 = 2, 1 = 3+ |
| Repeat/expansion revenue | 1 | 0 = none, 0.5 = some, 1 = significant |
