# Visual Identity — Strategic Questions & Build Pipeline

> 15 strategic questions for Build Mode + 9-step pipeline to transform answers into complete visual identity + child skills.

## Extract Mode: 5 Analysis Categories

When analyzing existing visual materials, systematically check:

### 1. Logo Patterns
- Logo variants in use (primary, secondary, icon-only)
- Dark mode / light mode versions
- Consistency across channels (website vs social vs print)
- Clear space and minimum sizing patterns
- Common misuses or violations

### 2. Color Patterns
- Primary colors used across all materials
- Secondary and accent color usage
- Color consistency or drift over time
- Accessibility: contrast ratios tested
- Competitor color overlaps (colors to avoid)

### 3. Typography Patterns
- Fonts in use (heading, body, accents)
- Hierarchy consistency (H1-H6 usage)
- Serif vs sans-serif patterns
- Pairing rules (when each font is used)
- Web vs print differences

### 4. Imagery Patterns
- Photography style (subjects, mood, composition)
- Illustration style (if used)
- Image treatment consistency (filters, overlays)
- Visual metaphors frequently used
- Stock photography vs custom

### 5. Component Patterns
- Button styles (colors, borders, shadows, hover states)
- Card/container styles
- Form element treatments
- Spacing and grid patterns
- Responsive behavior

**Extract output**: Document observed patterns → highlight inconsistencies → propose unified system.

## Build Mode: 15 Strategic Questions

Use when no consistent visual identity exists OR building from scratch.

### Block 1: Visual Personality (3 questions)

**Q1: Core Visual Traits**

What are 3-4 words describing how your brand should LOOK?

**Options**: Modern/classic, minimal/complex, bold/subtle, playful/serious, warm/cool, organic/geometric, clean/textured, structured/freeform

**Why this matters**: These traits guide ALL visual decisions. Everything else flows from this.

**Q2: Brand Voice Alignment**

How should the VISUAL identity express the brand VOICE personality?

[LOAD brand-voice AI Brand Kit for reference]

- What voice traits (from brand-voice) should visuals embody?
- If voice is "authoritative," how does that translate visually? (structured layouts, deep colors, serif fonts?)
- If voice is "rebellious," how does that translate visually? (unconventional, asymmetric, edgy?)

**Why this matters**: Visual and voice must be aligned. Misalignment confuses brand perception.

**Q3: Competitive Differentiation**

Visually, what do competitors look like? How should you differentiate?

- Describe 2-3 main competitors' visual aesthetics
- What visual clichés are overused in your industry?
- How can you stand out visually? (More colorful? More minimal? More human? More technical?)

**Why this matters**: Visual identity is positioning. Looking different = standing out.

---

### Block 2: Color Psychology (3 questions)

**Q4: Color Meaning**

What should your primary colors COMMUNICATE?

**Emotions/values to consider**: Trust (blue), energy (red/orange), growth (green), innovation (purple/cyan), luxury (black/gold), friendliness (yellow/orange), professionalism (navy/gray)

What emotion or value is most important for your brand to convey visually?

**Why this matters**: Color psychology is real. Wrong colors = wrong emotional response.

**Q5: Color Preferences**

- What colors do you strongly associate with your brand? (already using or naturally drawn to)
- Any colors to AVOID? (competitor colors, industry clichés, personal aversions)

**Why this matters**: Start with existing equity if available. Avoid colors that create wrong associations.

**Q6: Accessibility Commitment**

How important is WCAG 2.2 AA/AAA compliance?

- Legal requirement? (healthcare, government, education)
- Ethical commitment? (want to be inclusive)
- Nice to have? (optimize where possible)

**Why this matters**: Determines color palette constraints. WCAG AAA (7:1 contrast) is more restrictive than AA (4.5:1).

---

### Block 3: Typography Strategy (3 questions)

**Q7: Typography Personality**

Where do you sit on these spectrums? (1-5 scale)

- **Serif ↔ Sans-serif ↔ Display** (1 = serif only, 3 = sans-serif, 5 = display/decorative)
- **Classic ↔ Modern** (1 = traditional, timeless, 5 = cutting-edge, trendy)
- **Simple ↔ Distinctive** (1 = highly legible, safe, 5 = unique, memorable, bold)

**Why this matters**: Typography communicates personality before words are read.

**Q8: Readability vs Personality**

What's the priority?

- **Option A**: LEGIBILITY first (clean, safe, professional) — sacrifice uniqueness for clarity
- **Option B**: PERSONALITY first (unique, memorable, bold) — sacrifice some readability for distinctiveness
- **Option C**: Balanced (distinctive enough to be memorable, readable enough to be functional)

**Why this matters**: Determines font selection strategy. Headlines can be more distinctive; body text needs legibility.

**Q9: Typography Pairing**

- **One font family** for everything (uniform, simple, cohesive)
- **Heading + body pairing** (more visual interest, hierarchy)
- **Full type system** (heading, body, accent — complex but flexible)

**Why this matters**: More fonts = more complexity. Most brands need 1-2 fonts maximum.

---

### Block 4: Imagery Direction (3 questions)

**Q10: Primary Imagery Type**

Photography, illustration, abstract graphics, or mix?

**Photography**: Realistic, human, relatable, shows real products/people
**Illustration**: Flexible, customizable, distinctive, scales easily
**Abstract**: Conceptual, modern, metaphorical
**Mix**: Photography for some contexts (people, products), illustration for others (concepts)

**Why this matters**: Determines production requirements (photographer vs illustrator vs AI generation).

**Q11: Imagery Mood**

**If showing people**:
- Candid or posed?
- Diverse or specific demographic?
- Smiling/serious?
- Realistic/aspirational?
- Professional/casual?

**If abstract**:
- Calm/energetic?
- Organic/geometric?
- Colorful/monochromatic?

**Why this matters**: Mood consistency = brand recognition.

**Q12: AI Image Generation**

Will you use AI-generated imagery (Midjourney, DALL-E, Nanobanana)?

If yes:
- What style keywords feel right for your brand?
- Any styles to avoid?
- Will you combine AI + human editing (Photoshop post-processing)?

**Why this matters**: AI generation is a tool in the toolkit. If using it, need style prompts.

---

### Block 5: Visual World & Mapping (3 questions — NEW)

**Q13: Visual Universe**

What objects, scenes, or characters belong in your brand's visual universe?

Think of it like a movie: what are the recurring props, settings, and cast?

**Prompts**:
- "When you picture your brand's world, what objects do you see?"
- "If your brand was a physical place, what would be in it?"
- "What visual elements should appear repeatedly across your content?"

**Why this matters**: Foundation for Layer 1 Visual World. Defines what CAN appear.

**Q14: Visual Exclusions**

What should NEVER appear in your brand visuals?

**Prompts**:
- "What objects or imagery feel off-brand to you?"
- "What visual clichés does your industry overuse?"
- "Modern vs vintage? Corporate vs casual? What's wrong for you?"

**Why this matters**: Exclusions are as important as inclusions. Prevents off-brand elements.

**Q15: Content Type Visual Adaptation**

How should visuals adapt by content type?

**Prompts**:
- "Should blog hero images look different from social media graphics?"
- "Landing page visuals vs email visuals — same style or different energy?"
- "Technical content (docs, tutorials) vs marketing content (ads, posts) — different visual treatment?"

**Why this matters**: Foundation for Layer 2 Idea Mapping. Defines systematic approach per content type.

## Build Pipeline: Questions → 3-Layer System → Child Skills

### Step 1: Synthesize Visual Personality

**From**: Q1 (core traits) + Q2 (voice alignment) + Q3 (differentiation)

**Output**: 3-4 core aesthetic traits that:
- Align with brand voice personality
- Differentiate from competitors
- Guide all visual decisions

**Example**: "Clean modernism — generous white space, geometric shapes, no unnecessary decoration. Premium without being cold. Differentiates via simplicity in an overcomplicated industry."

**Validation**: Load brand-voice AI Brand Kit. Do these visual traits EXPRESS the voice traits?

---

### Step 2: Define Color System (WCAG 2.2 Tested)

**From**: Q4 (color meaning) + Q5 (color preferences) + Q6 (accessibility commitment)

**Process**:
1. Select primary color (1-2) based on meaning + preferences
2. Select secondary colors (1-2) for variety
3. Select accent colors (1-2) for CTAs/emphasis
4. Build neutral palette (4-6 grays + black + white)
5. Define semantic colors (success/warning/error/info)
6. **Test WCAG 2.2 compliance**: All text/background combinations
   - AA: 4.5:1 for normal text, 3:1 for large text
   - AAA: 7:1 for normal text, 4.5:1 for large text
7. Adjust colors if accessibility fails
8. Document color usage rules (when to use each)

**Output**: Complete color system with hex, RGB, CMYK, Pantone + WCAG ratings

**Validation**: All color combinations meet minimum accessibility (AA minimum, AAA preferred).

---

### Step 3: Set Typography Hierarchy

**From**: Q7 (typography personality) + Q8 (readability vs personality) + Q9 (pairing)

**Process**:
1. Select typeface(s) based on personality scores
   - Serif if classic/traditional (scores 1-2 on modern scale)
   - Sans-serif if modern/clean (scores 3-5)
   - Display if highly distinctive priority (score 4-5 on distinctive scale)
2. If pairing: choose complementary heading + body fonts
3. Define hierarchy: H1, H2, H3, H4, H5, H6, body, captions
   - Font family per level
   - Size, weight, line-height, letter-spacing
4. Set web-safe fallback stacks
5. Document pairing rationale

**Output**: Complete typography system with hierarchy + fallbacks

**Validation**: Type scale looks balanced. Hierarchy is clear at a glance.

---

### Step 4: Build Visual World (Layer 1)

**From**: Q13 (visual universe) + Q14 (exclusions) + brand context

**Process**:
1. Propose 5-7 object categories based on answers + industry
2. Show examples from similar brands (Brian Castle: desk, mug, notebook)
3. User validates, adds, removes
4. Define 2-3 typical scenes/settings
5. Add characters if applicable
6. List exclusions explicitly
7. Validate with test: "For topic [X], which objects would you use?" → verify AI selects appropriately
8. Write `visual-world.md`

**Output**: `references/visual-world.md` (Layer 1 complete)

**Iteration**: Typically 2-3 rounds to finalize inventory

---

### Step 5: Create Idea Mapping System (Layer 2)

**From**: Q15 (content type adaptation) + Visual World (Layer 1)

**Process**:
1. Load Visual World inventory from Step 4
2. Define decision tree: content type → core concept → visual approach (literal/metaphor) → objects from inventory
3. Build content type patterns (blog, landing page, social, course, technical)
4. Show example mappings from Brian Castle + add domain-specific
5. User validates, adds industry-specific mappings
6. Validate with tests: Give 3 random topics, verify AI selects appropriate visuals
7. Write `idea-mapping.md`

**Output**: `references/idea-mapping.md` (Layer 2 complete)

**Iteration**: 2-3 rounds to cover all content types

---

### Step 6: Establish Aesthetic Guidelines (Layer 3)

**From**: Q10 (imagery type) + Q11 (imagery mood) + Q12 (AI generation) + Q1-Q3 (personality)

**Process** (MOST time-intensive, 5-10 iterations):
1. Load brand-voice AI Brand Kit (alignment)
2. Propose 2-3 visual style directions based on answers
   - "Option A: Clean line art, minimal, modern"
   - "Option B: Bold comic book style, dramatic"
   - "Option C: Soft illustration, friendly, approachable"
3. **Generate 3-5 sample images** (Nanobanana) for each direction
4. Show samples: "Which resonates?"
5. User selects direction or requests variations
6. **Iteration loop**:
   - User: "Option B, but try thicker lines"
   - Regenerate with adjustment
   - User: "Good, but more colorful"
   - Regenerate
   - Repeat until: "This is it"
7. Codify final aesthetic:
   - All 7 dimensions (illustration style, color usage, line weight, detail, shadows, texture, background)
   - Build AI prompt library (base prompt + style keywords + negative prompts)
8. **Consistency test**: Generate 3 images with DIFFERENT subjects using final prompt
9. Verify all 3 look cohesive (same "world")
10. Write `visual-style.md`

**Output**: `references/visual-style.md` (Layer 3 complete) + AI prompt library

**Iteration**: 5-10 generations typical. User knows when it's right.

---

### Step 7: Generate Sample Visuals (Test Generation)

**Purpose**: Verify Layer 3 aesthetic works in practice before generating child skills.

**Process**:
1. Load visual-world.md (Layer 1)
2. Load idea-mapping.md (Layer 2)
3. Load visual-style.md (Layer 3)
4. Select 3 different topics: one literal, one metaphorical, one technical
5. Use Idea Mapping to select concepts
6. Use Aesthetic Guidelines to generate images
7. Review: Do all 3 feel like same brand? Pass 4 iteration criteria?

**Iteration Criteria Validation**:
- ✅ Flexible: All 3 different subjects look cohesive
- ✅ Distinctive: Doesn't look generic AI-generated
- ✅ Consistent: Matches brand voice personality
- ✅ Scalable: Style works across use cases

If any criterion fails → adjust Layer 3 → regenerate samples.

---

### Step 8: Create Visual Do's / Don'ts Pairs

**From**: All layers + sample generations

**Process**:
1. For each aesthetic dimension, create 1-2 paired examples
2. Format: ✅ Do This (on-brand) vs ❌ Not That (off-brand) + reasoning

**Examples**:

| ✅ Do This | ❌ Not That | Why |
|------------|-------------|-----|
| "Clean line art, 3px outlines, 2-color palette" | "Photorealistic rendering with complex shading" | Violates minimal/clean trait |
| "Workspace scene with 2-3 objects from Visual World" | "Random stock photo of generic office" | Violates Visual World constraint |
| "Bold sans-serif for headlines" | "Decorative script font" | Violates modern/clean trait |
| "High contrast, dramatic composition" | "Soft, subtle, low-contrast" | Violates bold trait |

**Target**: 5-10 pairs covering key decisions

**Usage**: Quality control for generated visuals. If output violates a "Don't," regenerate.

---

### Step 9: Define Component Library (Lightweight)

**From**: All previous steps + color/typography systems

**Components to define**:

**Buttons**:
- Primary button: Background color, text color, border, shadow, hover state
- Secondary button: Same specs
- Disabled button: Same specs

**Cards/Containers**:
- Background color
- Border (color, thickness)
- Shadow (offset, blur, opacity)
- Padding/spacing

**Forms**:
- Input field: Background, border, focus state
- Error state: Color, border treatment
- Label: Typography specs

**Navigation**:
- Menu items
- Active state
- Hover treatment

**Spacing Scale** (8pt grid recommended):
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- xxl: 48px

**Output**: Component specifications for `[brand]-ui-system` child skill

**Note**: This is LIGHTWEIGHT — just enough for web consistency, not a full design system.

## Brand Voice Alignment Table

Use this to ensure visual traits express voice traits:

| Brand Voice Trait | Visual Expression Examples |
|-------------------|---------------------------|
| **Playful** | Bright colors, rounded corners, friendly illustrations, dynamic compositions |
| **Authoritative** | Deep blues/grays, serif typography, structured layouts, formal imagery |
| **Innovative** | Bold colors, modern sans-serif, geometric patterns, tech-forward imagery |
| **Approachable** | Warm colors, open spacing, real people photography, soft shadows |
| **Premium** | Restrained palette, generous white space, elegant serif, refined imagery |
| **Rebellious** | Unconventional colors, asymmetric layouts, edgy imagery, break-the-rules composition |
| **Trustworthy** | Consistent colors, clear typography, professional photography, structured layouts |
| **Energetic** | Vibrant colors, dynamic compositions, motion-implied imagery, bold typography |

**Alignment Check Process**:
1. Load brand-voice 3 adjectives (e.g., "Bold, Direct, Practical")
2. For each adjective, identify visual expressions from table
3. Ensure aesthetic specs (color, typography, imagery) embody those expressions
4. If misalignment: adjust aesthetic to better express voice

**Example**:
- Voice trait: "Bold"
- Visual expressions: Bold colors, strong typography, high contrast
- Check aesthetic specs: Do we have bold colors? ✓ Strong typography? ✓ High contrast? ✓
- Aligned ✅

## Extract vs Build Decision Matrix

| Existing Materials | Visual Snapshot Confidence | Recommendation |
|-------------------|---------------------------|----------------|
| Comprehensive (website, brand guide, extensive content) | High | Extract mode (Fast, respects equity) |
| Some materials (website only, no guide) | Medium | Hybrid (Extract what exists, Build gaps) |
| Minimal (logo only, pitch deck) | Medium-Low | Hybrid (Extract logo/colors, Build rest) |
| None (pre-launch, rebranding) | Low | Build mode (Create from scratch) |

**Extract mode**: Use existing as foundation, codify and systematize
**Build mode**: Answer all 15 questions, create from strategy
**Hybrid**: Extract what's strong, Build what's missing

## Iteration Guidelines

### For Layer 3 Aesthetic (Expect Most Iterations)

**Early iterations** (rounds 1-3):
- Exploring different style directions
- Broad changes ("more minimal" vs "more detailed")
- Finding general aesthetic family

**Mid iterations** (rounds 4-7):
- Refining within chosen direction
- Specific adjustments ("thicker lines", "more color")
- Tuning detail level and texture

**Late iterations** (rounds 8-10):
- Fine-tuning
- Small tweaks ("slightly less saturated")
- Locking in final aesthetic

**How to know you're done**:
- User says "This is it" or "Perfect"
- Generated samples consistently pass 4 criteria
- No more "but try..." feedback

Don't rush this. The aesthetic locked here will generate hundreds/thousands of images via child skills.
