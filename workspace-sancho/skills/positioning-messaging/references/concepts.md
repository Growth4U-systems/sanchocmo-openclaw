# Positioning & Messaging — Concepts & Methodology

## Core Concept

Per-niche positioning: one framework per ECP, not one generic pitch. The full 7-step pipeline runs for EACH selected ECP, starting with highest-priority.

## Tier 2 Document Management

Four Tier 2 documents shared across niches:

| Document | Scope | Rule |
|----------|-------|------|
| **Value Criteria** | Shared | Check existing before creating. Add only new. Never duplicate. |
| **Assets** | Shared | Check existing before creating. Same asset may connect to different criteria per niche. |
| **Niche Profiles** | Per ECP | References shared criteria and assets |
| **Messaging Playbooks** | Per ECP | References shared criteria |

**These form related databases**: Value Criteria ↔ Assets ↔ Niches ↔ Messaging.

**When processing a NEW niche:**
1. Load existing Value Criteria → check matches → add only new
2. Load existing Assets → check matches → add connections, not duplicates
3. Create new Niche Profile referencing existing + new
4. Create new Messaging Playbook referencing criteria

---

## Value Criteria Discovery (Step 4)

Criteria must blend functional + emotional drivers across 5 dimensions:
- **Compliance & Reliability**
- **Operational Efficiency**
- **Financial Intelligence**
- **Ecosystem Connectivity**
- **Emotional Drivers**

Naming: short Noun Phrases (2-5 words), no codes, same abstraction level.

Scoring (0-5):
- **Red Ocean** (avg 4-5): High competition, needs well-satisfied
- **No Market** (avg 0-1): Low customer value or untapped
- **Opportunity Zone** (avg 2-3): Key for differentiation

---

## Asset Categorization (Step 5)

- **Qualifiers**: Market standard, must-have to play
- **Differentiators**: Unique, build business around

Rules: Avoid vague words ("empower", "elevate"). Niche down early. Be specific.

---

## Messaging Principles (Step 7)

- 1 row for UVP Core Promise + 4-5+ rows for USPs
- Short, sharp, emotionally intelligent copy
- Benefit-driven over feature-driven
- Pain points and solutions obvious within 1 sentence
- Each message connected to a Value Criterion
- Bilingual (EN + ES), natively written (not translated)

---

## Lite vs Deep Criteria

**Lite** (per niche): Steps 1-4 + 3 assets mapped + UVP + 3 USPs bilingual
**Deep** (per niche): All 7 steps + full scoring map + complete assets + full proof table + complete bilingual playbook + Tier 2 updated

---

## Edge Cases

- **No proof for key claims**: Flag. Don't message around unproven claims.
- **All competitors say the same**: Look for Opportunity Zone angles nobody talks about.
- **Client wants to position on weakness**: Redirect with Lens 3 data.
- **Single ECP**: Valid, especially early-stage. Run full 7 steps.
- **Multi-language**: Core positioning stays same. Messaging adapts natively.
- **Existing Tier 2 covers everything**: Skip new criteria/assets, just create playbook.
