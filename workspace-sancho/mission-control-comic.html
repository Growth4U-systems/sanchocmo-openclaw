Crear proyecto Supabase

        
Base de datos operacional del cliente (9 tablas).

        

          
☐Ir a Supabase Dashboard

          
☐"New Project" → nombre: sanchocmo-{slug}

          
☐Copiar Project URL y anon key (Settings → API)

          
☐SQL Editor → pegar y ejecutar init-db.sql 

          
☐Verificar 9 tablas en Table Editor

        

        

          

        

      


      
      

        
Configurar Discord

        
Servidor con canales para los agentes.

        

          
☐Discord → "+" → Crear servidor → nombre: SanchoCMO — {nombre}

          
☐Crear 5 categorías + 14 canales:
            

              📁 ESTRATEGIA: #el-toboso, #campaigns, #intelligence

              📁 OUTREACH: #partners, #prospecting

              📁 CONTENT: #organic-content, #social, #paid-ads, #web

              📁 SOPORTE: #sales, #design, #research

              📁 SISTEMA: #learning, #admin
            

          

          
☐Invitar bot SanchoCMO al servidor (usar URL de OAuth2 existente)

          
☐Copiar Guild ID (Server Settings → Widget o Developer Mode → Copy Server ID)
